В.К.Л — бот помощник

Интерактивный Telegram бот для клана Clash of Clans — следите за войнами, топами, участниками и столицей клана!

Открыть бота в Telegram

Команды бота

Используйте команды в чате с ботом:

  • /info — информация о клане
  • /members — список участников (TH, кубки, роль)
  • /top — топ игроков по кубкам и донатам
  • /war — информация о текущей войне
  • /capital — столица клана
  • /donations — топ донатеров
  • /stats — сводная статистика

О боте

Бот написан на Python с использованием python-telegram-bot. Получает актуальные данные через официальное API Clash of Clans, чтобы вы всегда знали состояние клана и его участников.

Интерактивность

Бот умеет показывать текущую войну, топ-игроков, донаты и столицу клана. Добавляйте бота в свои группы и следите за кланом вместе с друзьями!
